# Knowji

A Nuxt 3 library for Knowji.

## Features

- Built with Nuxt 3
- TypeScript support
- Vue 3 components
- Modern development tools

## Requirements

- Node.js >= 18.20.6
- npm >= 9.0.0

## Installation

```bash
npm install knowji
```

## Usage

Add to your Nuxt 3 project:

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['knowji']
})
```

## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Building

```bash
npm run build
```

## Linting

```bash
# Check code style
npm run format:check

# Fix code style
npm run format

# Lint code
npm run lint

# Lint and fix
npm run lint:fix
```
